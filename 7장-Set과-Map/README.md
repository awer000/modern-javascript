### Set

순서가 있는 유일한 값의 리스트  
자동으로 중복값을 제거함  
배열의 Sub Class 는 아니다.

### WeakSet

객체만 포함할 수 있는 특별한 Set  
객체는 약한 참조로 저장되고 WeakSet 내의 요소가 객체에 대한 참조로만 남아 있다면 가비지 컬렉션 대상이 됨

### Map

순서가 있는 키, 값 쌍의 리스트  
어떤 타입이든 키가 될 수 있다.

### WeakMap

객체 키만 가질 수 있음  
객체 키 참조는 약한 참조이고, 키가 가비지컬렉션 될 때, 키와 연관된 값 또한 WeakMap 에서 제거된다
